HTTP (HyperText Transfer Protocol, или **Протокол передачи гипертекста**)- набор правил взаимодействия между клиентом и сервером в  Интернете. 
HTTP используется для передачи HTML-страниц, изображений, видео, текстов и других типов данных, составляющих веб-ресурсы.
HTTP использует соединение [[TCP]] (или [[TLS]]-защищённый TCP)
