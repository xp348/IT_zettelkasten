OSI (Open Systems Interconnection) - описание взаимодействие открытых систем. Это эталонная модель которая описывает как данные должны передаваться по сети.
Уровни передачи данных:
1. Физический уровень L1 (Physical Layer)
2. Канальный уровень L2 (Data Link Layer)
3. Сетевой уровень L3 (Network Layer)
4. Транспортный уровень L4 (Transport Layer)
5. Сеансовый уровень L5 (Session Layer)
6. Представительный уровень L6 (Presentation Layer)
7. [[Прикладной уровень (L5 – L7) сетевого взаимодействия|Прикладной уровень L7]] (Application Layer)

Модель OSI  не нашла широкого применения. на данный момент интернет работает по TCP/IP.

TCP/IP модель почти полностью повторяет эталонную OSI, но имеет свои особенности и имеет 5 уровней:
1. Канальный (объединяет L1 и L2 из OSI)
2. Межсетевой (идентичен  L3)
3. Транспортный (идентичен  L4)
4. Прикладной (объединяет L5 – L7)
